action(object, deferred)

点击按钮执行的自定义操作
参数:
Name Type Description
object Object 当前页的值(recordNo起始于1)
deferred $.Deferred 延时对象。函数中必需执行deferred.resolve()、deferred.reject()
Example
判断是否有选中记录
action:function(object,deferred){
   if(!object.recordNo)
       d.reject('请先选择一行');
   else if(object.pageRecords[object.recordNo-1].weight<1000)
       d.reject('重量小于1000');
   else{
       d.resolve();
       alert('操作成功');
   }
}
