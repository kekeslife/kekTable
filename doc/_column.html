Methods

afterChange(tableValues, defer)

栏位改变值以后。必需要写d.resolve('将要填入的值')或者d.reject('错误框文字')
参数:
Name Type Description
tableValues object.<colEntity> 该行的数据
defer $.Deferred deferred对象,d.reject()将停止事件,d.resolve()将继续执行。
返回值:
[['empNo','11111'],['empName','xxxxx']] - 将要连动更改的栏位和值。

beforeGetValue(tableValues, defer)

栏位读取(显示)数据之前的自定义操作。必需要写d.resolve('将要显示的文字或HTML，非数据库栏位的curPageRecords将保存text')或者d.reject('将会产生提示框')
参数:
Name Type Description
tableValues object.<colEntity> 该行的数据
defer $.Deferred deferred对象,d.reject()将停止事件,d.resolve()将继续执行。
Examples
ajax

	var xx=$.ajax('test.html').done(function(res){d.resolve(res.EmpName);}).fail(function(){d.reject('失败');});
